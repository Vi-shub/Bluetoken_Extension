# Security policy

## Reporting a vulnerability

BlueToken only reads **local** editor databases on your machine. It does not upload chat content.

If you find a security issue (e.g. unintended network calls, path traversal, or sensitive data written to logs):

1. Prefer a **private** report (email / contact form on [shubhamvishwakarma.com](https://www.shubhamvishwakarma.com)) rather than a public GitHub issue.
2. Include IDE, OS, BlueToken version, and steps to reproduce.
3. Do **not** attach real chat transcripts.

We aim to acknowledge reports within a few days.

## Supported versions

| Version | Supported |
|---|---|
| Latest `0.1.x` on `main` | Yes |
| Older VSIX builds under `releases/` | Best-effort |
